# Grunt DevStack

Grunt Dev Stack with less and bable.

---

## Instalation

1.  Download or clone repository
2.  run ```npm install```
3.  run ```bower install```

## Grunt tasks

1. ```grunt``` – for development.
2. ```grunt build``` – for production.
3. ```grunt build-image``` – for optimize images.
