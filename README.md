# jsSwipeDetection

JavaScript for detection of swipe direction.

---

## Instalation

Download or clone repository.

## NPM

npm install js-swipe-detect

## Devel
```npm install``` – for development.

## Grunt tasks

1. ```grunt``` – for development.
2. ```grunt build``` – for production.
